Padding is the space between the content and border, It can be used to create space around an element's content inside a defined border by setting the top,bottom,left and right value.
Border allows us to specify the style,size, width and color of an element's border.
Margin is used to define a space around the elements ,We can do this by setting the margin style to auto,or setting a style using top and bottom.

Padding,Border and Margin are a part of the box model,box model is a box that wraps around every HTML element.
